📁 How to Use These Codes

These activation codes are VIN-specific and verified against the PCM 3.1 firmware's CPPorscheEncrypter class. To activate features:

  1. Format a USB stick as FAT32 (4GB–32GB)
  2. Create the activation file with codes (format documentation in progress)
  3. Insert USB into the PCM's USB port
  4. PCM auto-detects and processes activation in 30–60 seconds
  5. PCM reboots with features enabled

How it works: Copy all 3 files to a FAT32 USB stick root: copie_scr.sh (XOR bootstrap), run.sh (plaintext script), and PagSWAct.002 (activation codes). The Save to folder button also bundles bin/showScreen and lib/*.png status screens automatically. Insert into Cayenne USB port with ignition on. Wait 60–90 seconds. Remove and cycle ignition.
Replaces copie_scr.sh with an enhanced diagnostic probe that writes pcm_debug.log + pcm_dump/ folder back to USB — 29+ tests: PCM version, VIN, activation files, variant coding (CVALUE), FSC state, CAN bus, network/telnet, flash & HDD partitions, engineering ESD screens, browser paths, HMI region, SWDL state, HDD SMART data, plus CAN/IOC probe (IPC devices, DSP, FPGA sysregs, MOST, service broker, inetd.conf). Backs up existing PagSWAct. No changes are made to the car.

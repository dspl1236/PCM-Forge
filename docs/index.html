Build activation USB stick
How it works: Copy all 3 files to a FAT32 USB stick root: copie_scr.sh (XOR bootstrap), run.sh (plaintext script), and PagSWAct.002 (activation codes). Insert into Cayenne USB port after the PCM has fully booted (wait for the home screen). Do NOT insert before starting the car — the PCM only detects scripts on a fresh USB insert event. Wait 60–90 seconds, then remove USB and hard reboot the PCM: press and hold INFO + CAR simultaneously until the screen goes black.
⚠ Important: The PagSWAct.002 file replaces all existing activations on your PCM. Only the features you select below will be active after installation — anything not selected will be deactivated.
Recommended workflow:
1. Run Diagnostic mode first to back up your current activations
2. Upload your backup in the Backup tab to see what’s currently active
3. Use Select all to keep all existing features, then add any new ones
4. Build your USB with the complete feature set

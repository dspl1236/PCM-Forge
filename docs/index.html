Enhanced 29-test diagnostic probe — no changes are made to the car. Writes pcm_debug.log + pcm_dump/ folder to USB with PCM version, VIN, variant coding (CVALUE), CAN bus, network/telnet, flash & HDD partitions, engineering screens, HMI region, SWDL state, HDD SMART data. Also backs up your current PagSWAct.002 to the USB stick as PagSWAct_backup.002. No features needed — just check this box and write to USB.
After you run the diagnostic USB 1. Insert USB with ignition on, wait 60–90 sec.
2. Remove USB, put it back in your PC.
3. You should see pcm_debug.log on it (and maybe pcm_ran.txt). If pcm_debug.log is missing, the PCM never ran the script — tell us that.
4. Drag the log into the chat. We'll read it and figure out the next move.
